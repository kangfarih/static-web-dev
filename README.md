# STATIC WEB DEV TEMPLATE

### Template for Developing Static Web

## How to use

### 1. Install NodeJs 
### 2. Install Degit
#### ```npm i -g degit```
### 3. Degit this repository
#### ```degit https://github.com/kangfarih/static-web-dev.git [folder_name]```
### 4. Go To [folder_name]
#### ```cd [folder_name]```
### 5. Install all dependency
#### ```npm install```
### 6. Start your development
#### ```npm run dev```
